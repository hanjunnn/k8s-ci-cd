✏️ 서비스 내용

  • 블록체인 기술을 활용하여 음원 제작자와 소비자 간의 직접적이고 안전한 거래를 지원하는 플랫폼을 개발함.
  • 사용자 간의 음원 거래 시 포인트와 현금을 관리하고, 이를 블록체인에 기록하여 거래의 투명성을 보장함.

🖥️ 서비스 화면 예시

🙋‍♂️ 담당 업무 (Front End & Back End)

  • 전반적인 html,css와 node.js를 통한 rest api 개발

📖 담당 개발 내용

  • 주요 페이지 구성
    • 설계시 디자인한 웹페이지를 HTML과 CSS를 사용하여 구성
  • 환불 기능 구현
    • 사용자가 선택한 금액을 환불하는 기능을 구현. Node.js를 이용한 REST API 엔드포인트 /refund에 POST 요청
  • 사용자 데이터 업데이트 및 새로고침
    • 데이터를 업데이트 하는 동작 후, 사용자의 최신 데이터를 가져오기 위해 /query 엔드포인트에 GET 요청을 보냄.
    • 요청된 데이터를 받아 userData를 업데이트하고, 페이지를 새로고침하여 동적으로 변경 사항을 반영.
