Tech Stack

이 프로젝트는 다양한 기술 스택을 활용하여 배포 및 관리되었습니다. 아래는 사용된 주요 기술들입니다.

Host OS & Virtual Machine

Host OS: Windows
프로젝트에서 호스트 운영 체제로 Windows OS를 사용하여 가상화 환경을 구축했습니다. Windows는 Hyper-V를 통해 가상 머신을 실행하는 데 적합한 환경을 제공합니다.

Virtual Machine: Hyper-V
Windows OS에서 Hyper-V를 사용하여 가상화 환경을 설정하고, 여러 가상 머신을 효율적으로 관리할 수 있었습니다. Hyper-V는 가상 머신을 실행하고 자원을 최적화하는 데 유용한 도구입니다.

Master & Worker Nodes: Rocky Linux
Rocky Linux는 프로젝트의 마스터 및 워커 노드 운영 체제로 사용되었습니다. 이 배포판은 안정성과 성능이 뛰어나 Kubernetes 클러스터 환경에서 최적의 성능을 발휘하며, 서버 및 클러스터 환경 운영에 적합합니다.

Kubernetes

Kubernetes는 컨테이너화된 애플리케이션을 자동으로 배포하고 관리하는 오픈 소스 시스템입니다. 이 프로젝트에서는 RKE2 (Rancher Kubernetes Engine 2)를 사용하여 Kubernetes 클러스터를 구축했습니다. RKE2는 안정성과 보안성을 강조한 Kubernetes의 경량화된 버전으로, 다양한 환경에서 쉽게 배포할 수 있습니다. 또한, Helm을 사용하여 애플리케이션을 Kubernetes 클러스터에 자동으로 배포하고 관리했습니다.

Helm은 Kubernetes의 패키지 관리 도구로, 애플리케이션을 Chart라는 템플릿으로 정의하여 배포하고 관리할 수 있습니다. Helm을 통해 복잡한 설정을 간단하게 패키징하여 Kubernetes 클러스터에 배포할 수 있으며, 다양한 환경에 맞춰 구성 파일을 쉽게 수정할 수 있습니다. 이 프로젝트에서는 Helm을 활용해 여러 애플리케이션을 효율적으로 관리하고, 반복 가능한 배포를 실현했습니다.

CI/CD Tools

  • Jenkins: CI/CD 파이프라인 자동화를 위한 Jenkins 사용. Jenkins는 애플리케이션 빌드, 테스트 및 배포를 자동화하여 개발자의 생산성을 높여줍니다.
  • ArgoCD: GitOps 기반의 지속적인 배포 및 관리 도구. ArgoCD는 Git 저장소를 배포 소스로 사용하여 Kubernetes 클러스터에 애플리케이션을 배포하고 관리하는 데 도움을 줍니다. GitOps 접근 방식으로 인프라를 코드화하여 더 나은 배포와 버전 관리를 실현할 수 있습니다.

Application Services

  • Cert Manager: SSL 인증서를 자동으로 관리하고 발급하는 도구입니다. Cert Manager는 Kubernetes 클러스터 내에서 인증서를 생성하고 갱신하며, 애플리케이션의 보안을 강화합니다.
  • Nginx: Nginx는 웹 서버이자 리버스 프록시 역할을 하는 소프트웨어로, 이 프로젝트에서는 Nginx를 사용하여 웹 트래픽을 관리하고, 애플리케이션에 대한 요청을 적절한 서비스로 라우팅했습니다.
    Kubernetes 환경에서는 Nginx를 Ingress Controller로 사용하여 외부 트래픽을 내부 서비스로 라우팅하고, Ingress 리소스를 통해 다양한 도메인과 경로를 하나의 인프라에서 처리할 수 있습니다. 또한, SSL/TLS 인증서와 같은 보안 요소를 설정하여 클러스터 내에서 안전한 트래픽 관리가 가능합니다.
  • Rancher: Kubernetes 클러스터 관리 및 오케스트레이션 도구입니다. Rancher는 여러 Kubernetes 클러스터를 중앙에서 관리할 수 있도록 하여, Kubernetes 환경을 보다 쉽게 관리하고 모니터링할 수 있도록 도와줍니다.

이 기술 스택들을 사용하여 로컬 VM 환경에서 Kubernetes 클러스터를 구축하고, 다양한 DevOps 툴을 통해 CI/CD 파이프라인을 자동화하며, Kubernetes 환경에서 애플리케이션을 안정적으로 배포하고 관리할 수 있었습니다. 이러한 기술들은 효율적인 배포와 안정적인 운영을 가능하게 만들어, 프로젝트의 성공적인 진행에 큰 기여를 했습니다.
