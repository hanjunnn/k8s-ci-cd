What's Kubernetes?

쿠버네티스(Kubernetes)는 컨테이너화된 애플리케이션의 배포, 확장 및 관리 자동화를 돕는 오픈소스 시스템입니다. 클러스터 환경에서 애플리케이션을 안정적으로 운영할 수 있도록 도와주며, 고가용성 및 자동화된 배포를 지원합니다.


이 사이트는 쿠버네티스를 통해 배포되었으며, 로컬PC에 VM 환경을 구축하여 쿠버네티스를 배포하고 프로젝트를 구현했습니다.
실습 과정에서는 Nginx를 웹서버로 사용하고, Nginx Ingress를 활용해 외부 트래픽을 관리하며,
Rancher를 사용해 쿠버네티스 클러스터를 관리했습니다.
또한, GitHub, Docker Hub, Jenkins, ArgoCD와 같은 다양한 도구들을 통합하여
자동화된 CI/CD 파이프라인을 구축하고, 애플리케이션의 지속적인 통합 및 배포를 구현했습니다.

Nginx

Nginx는 고성능 웹 서버이자 리버스 프록시 서버로, 주로 HTTP와 HTTPS 트래픽을 처리합니다. 또한, 로드 밸런싱과 캐싱 기능을 제공하며, 고용량의 웹 애플리케이션을 안정적으로 지원하는 데 사용됩니다.

Rancher

Rancher는 Kubernetes 클러스터를 관리하고 오케스트레이션하는 오픈소스 플랫폼입니다. 이 플랫폼은 여러 클러스터를 중앙에서 관리하고, 보안, 네트워킹, 스토리지, 로그 관리 등을 쉽게 처리할 수 있도록 도와줍니다.

Jenkins

Jenkins는 지속적 통합(CI)과 지속적 배포(CD)를 자동화하는 오픈소스 도구입니다. 다양한 플러그인을 통해 빌드, 테스트, 배포 작업을 자동화하며, 개발 프로세스를 빠르고 안정적으로 만들 수 있습니다.

  • Dashboard

ArgoCD

ArgoCD는 Kubernetes 네이티브의 지속적 배포(CD) 도구로, GitOps 방식으로 애플리케이션을 배포하고 관리합니다. Git 리포지토리를 기준으로 자동으로 배포와 동기화를 진행하여 애플리케이션의 상태를 관리합니다.
