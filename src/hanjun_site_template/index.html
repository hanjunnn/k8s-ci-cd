안녕하세요! 한준입니다.

이 사이트는 제가 진행한 프로젝트를 소개하고, 제가 사용한 기술 스택을 공유하기 위해 만들어졌습니다.
저는 클라우드 인프라 구축과 쿠버네티스, CI/CD 파이프라인을 활용한 자동화된 배포 프로세스를 구축하는 것에 관심이 많습니다.
이 사이트는 쿠버네티스를 통해 배포되었으며, 로컬PC에 VM 환경을 구축하여 쿠버네티스를 배포하고 프로젝트를 구현했습니다.
실습 과정에서는 Nginx를 웹서버로 사용하고, Nginx Ingress Controller를 활용해 외부 트래픽을 관리했습니다.
또한, Rancher를 사용해 쿠버네티스 클러스터를 관리하고, GitHub, Docker Hub, Jenkins, ArgoCD와 같은 다양한 도구들을 통합하여
자동화된 CI/CD 파이프라인을 구축하고, 애플리케이션의 지속적인 통합 및 배포를 구현했습니다.
